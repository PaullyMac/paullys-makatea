# Paully's Maka-tea
This project is an activity for the Web Development subject by **Paul Angelo Macaraeg**.

## Overview
Paully's Maka-tea is a responsive website for a premium tea business featuring a modern, user-friendly interface with an elegant gold and brown color scheme. The site provides a comprehensive view of the company's products, background, and ambiance through multiple interactive sections.

## Features
- **Home Section**: Company information, mission statement and contact details
- **Product Catalog**:
  - 9 premium tea varieties with images
  - Detailed product descriptions with origins and flavor profiles
  - Interactive product cards with modal popups
  - Price information for B2B inquiries
- **Music Player**:
  - Custom-built tea house jazz playlist
  - Video player with controls
  - Shuffle functionality and track selection
  - Volume control and auto-play features
- **Jobs Section**: Application form for career opportunities with validation
- **Responsive Design**: Fully responsive layout optimized for all devices from mobile to desktop
- **Tea-Themed Interface**: Elegant styling with organic textures and subtle animations

## How to Run
To run this project locally, follow these simple steps:

1. **Clone the repository**
   ```bash
    git clone https://github.com/PaullyMac/paullys-makatea.git 
    cd paullys-makatea
    ```

2. **Open the project**
- Simply open the index.html file in your web browser
- No server or build process required

3. **Alternative method using VS Code**
- Open the project folder in Visual Studio Code
- Install the "Live Server" extension
- Right-click on index.html and select "Open with Live Server"

## Technologies Used
